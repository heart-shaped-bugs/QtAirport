# Aircraft & Passenger Manager

[![Qt](https://img.shields.io/badge/Qt-6.10.2-blue.svg)](https://www.qt.io/)
[![C++](https://img.shields.io/badge/C++-17-green.svg)](https://isocpp.org/)

Приложение для управления авиакомпанией, разработанное на Qt/C++. Позволяет управлять флотом самолётов и базой пассажиров с возможностью сохранения данных в формате XML.

## О проекте

Это учебное/демонстрационное приложение, которое показывает возможности фреймворка Qt для создания десктопных приложений с графическим интерфейсом. Проект идеально подходит для изучения:
- ООП на C++
- Qt Widgets и сигналов/слотов
- Работы с табличными данными (QTableWidget)
- XML сериализации данных
