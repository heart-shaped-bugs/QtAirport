# QtAirport
Aircraft &amp; Passenger Manager

Приложение для управления авиакомпанией на Qt/C++. Позволяет отслеживать самолёты и пассажиров, сохранять данные в XML файлах.

## Возможности

-  Управление флотом самолётов (модель, вместимость, авиакомпания)
-  Управление базой пассажиров (имя, фамилия, номер рейса)
-  Сохранение данных в XML формате
-  Загрузка данных из XML файлов
-  Добавление, удаление и редактирование записей
-  Интуитивный графический интерфейс с двумя вкладками

## Технологии

- **Qt 6** (Widgets, XML)
- **C++17**
- **QVector** для хранения данных
- **QDomDocument** для работы с XML
- **CMake** для сборки
